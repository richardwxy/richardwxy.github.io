需求：实现一个类似街景地图360°环绕的效果。
工具：JavaScript和three.js。
（three.js 是基于WebGL开发的 可以使用js来编写3D程序）

那么three.js包含几个部分呢？应该怎么使用呢？

1.three.js渲染器：
     THREE.WebGLRenderer 负责设置画布大小，是否抗锯齿，背景颜色等属性。
2.three.js相机：
    camera 有透视相机和正投影相机两种。(详情)
     THREE.PerspectiveCamera 
     THREE.OrthographicCamera  
     本例选取透视相机。
3.three.js场景：
     THREE.Scene 场景就是一个三维空间。
4.three.js光源：
     光源分为：
     THREE.AmbientLight 环境光
     THREE.DirectionalLight 平行光
     THREE.PointLight 点光源
     THREE.SpotLight 聚光灯 等光源
     本例选取环境光。
5.three.js 3D对象：
     这个看需要什么 比如需要显示一个球体 可以使用 THREE.SphereGeometry。


参考官方demo cubemap 关于贴图顺序参考可查询cubemap 其中：
                var path = "textures/cube/SwedishRoyalCastle/";
                var format = '.jpg';
                var urls = [
                        path + 'px' + format, path + 'nx' + format,
                        path + 'py' + format, path + 'ny' + format,
                        path + 'pz' + format, path + 'nz' + format
                    ];

                var reflectionCube = new THREE.CubeTextureLoader().load( urls );
                reflectionCube.format = THREE.RGBFormat;

                scene = new THREE.Scene();
                scene.background = reflectionCube;
可实现360贴图。
